Requirements
------------
jack56c
WDAS Core

Source code
------------
src/main: contains support classed for the agents and the main entry point
src/jack: contains jack agents. Use JDE to edit and compile.
src/test: contains unit tests for the agents and related classes
src/resources: contains additional files used from within the agents or related classes
